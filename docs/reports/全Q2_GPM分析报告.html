1 执行摘要

本报告对小红书商品笔记项目在2026年3月至本周（约7月初）共五个周期的GPM表现进行全面分析。Q2整体呈现出典型的"震荡冲高—见顶回落—尾端反弹"三阶段走势：

核心结论：整体GPM在3~5月于35~55区间反复震荡，4月达到GMV峰值的¥89,390；6月起进入缩量调整期，曝光和GMV同步收缩；本周受分母效应（曝光量骤降至9.1万，仅为5月峰值的4.4%）推动，GPM飙升至131.93，创Q2新高。四位核心创作者中，鲜水湾表现最为稳健（3-6月均值59.2），小柚成长速度最快（3月22.44→本周114.98，增长412%），而视频笔记GPM波动剧烈（5月峰值77.55→本周归零），变现稳定性亟待改善。

基于五期数据的五大关键发现：

  1. GPM震荡上行但基础不稳：前4期在34~55区间运行，本周因子弹效应飙升至131.93，需警惕数据异常对判断的干扰。
  2. GMV在4月见顶后持续萎缩：¥89,390→¥43,438→¥11,997，降幅达86.6%，商品变现热度显著降温。
  3. 创作者梯队分化明确：鲜水湾（稳健型）vs 小柚（成长型）vs 菲菲（波动型）vs 婉婉（平稳型），各具不同运营价值。
  4. 视频变现极不稳定：从5月视频GPM 77.55的巅峰急转至本周归零，单条爆款依赖度过高。
  5. 低曝光高GPM笔记集中涌现：多位创作者的"69元虾仁"内容呈现超强转化力，曝光量虽低但GPM普遍在200~2,800区间。

2 数据概览与方法论

2.1 数据来源与范围

本报告分析的数据涵盖5个时间周期的商品笔记表现数据：

  • 3月数据：2026年3月全月，匹配笔记641条，有GMV笔记143条
  • 4月数据：2026年4月全月，匹配笔记990条，有GMV笔记206条
  • 5月数据：2026年5月全月，匹配笔记1,477条，有GMV笔记239条
  • 6月数据：2026年6月全月，匹配笔记1,805条，有GMV笔记172条
  • 本周数据：最近一周（约6月底~7月初），匹配笔记1,843条，有GMV笔记72条

数据来源为小红书商品笔记运营后台，经数据科学团队提取、清洗和标准化处理。

2.2 核心指标定义

  • GPM（千次曝光成交额） = GMV ÷ （曝光量 ÷ 1000）。代表每千次曝光产生的成交金额，是衡量内容变现效率的黄金指标。
  • GMV：商品成交总额，反映整体变现规模。
  • 曝光量：笔记被用户看到的次数，反映内容分发广度。
  • 客单价 = GMV ÷ 有GMV笔记数，反映单笔记平均成交价值。
  • 转化率 = 有GMV笔记数 ÷ 匹配笔记总数，反映内容到成交的转换效率。

2.3 分析方法

  • 趋势分析：五期关键指标的纵向对比，识别长周期规律
  • 创作者分群：按GPM表现和波动特征对四位核心创作者聚类
  • 异常检测：对视频GPM剧烈波动的根因分析
  • 专题分析：低曝光高GPM笔记的共性提取
局限性说明：本周数据为部分周期（约1周），与整月数据对比时存在时间窗口不对等的问题。GPM本周飙升部分受"分母效应"（曝光量骤降）影响，需客观看待。视频笔记样本量在各周期分布不均，结论仅供参考。

4 月维度深度分析

4.1 3月：冷启动期（GPM 37.19）

3月是整个项目周期的起点月，各项指标处于起步阶段：

  • 匹配笔记641条、有GMV笔记143条，变现转化率22.3%
  • 总GMV ¥22,259，客单价¥88.33，整体GPM 37.19
  • 视频GPM仅9.77，视频笔记变现效率极低，几乎为图文笔记的补充角色
  • 曝光总量不足60万，处于内容分发的初始积累期
关键特征：3月是"验证期"——用较少的笔记测试市场和内容方向，GPM基数适中，视频能力尚未建设。此时整体GPM以图文笔记为绝对主力。

4.2 4月：爆发期（GPM 54.10，GMV峰值¥89,390）

4月是Q2的黄金月，各项指标全面爆发：

  • GMV飙升至¥89,390，较3月增长301.6%，创全Q2峰值
  • 曝光量增长至1,652,414（+176%），GPM提升至54.10（+45.5%）
  • 客单价跃升至¥124.15（+40.6%），说明高价值商品开始获得成交
  • 视频GPM从9.77跃升至56.88（+482%），视频变现能力大幅改善
  • 有GMV笔记206条，达到全Q2第二高的水平
成功因素：4月实现了"量价共振"——曝光量增长驱动了更多成交，同时GPM的同步提升说明内容质量和精准度也在改善。这是Q2最健康的运营月份。

4.4 5月~本周：调整收缩期

5月起各项指标进入持续收缩通道：

  • 5月：GMV ¥79,650（-10.9% vs 4月），但曝光量增至2,050,217（+24.1%），GPM回落至38.85。视频GPM达到峰值77.55，但整体效率走低。
  • 6月：GMV进一步缩至¥43,438（-45.5%），曝光量降至1,254,571（-38.8%）。GPM降至34.62的低点。
  • 本周：GMV仅¥11,997，曝光量骤降至90,932（仅为5月峰值的4.4%），但因分母效应GPM飙升至131.93。
核心警示：5月曝光量虽创新高但GPM持续下降，说明内容分发的精准度在恶化——更多曝光并未带来同比成交增长，可能存在内容与目标用户匹配度下降的问题。

6.1 演进路径：从9.77到77.55再到0

异常信号：视频GPM从3月的9.77飙升至5月的77.55（增长694%），又在6月骤降至19.25，本周归零。这一"过山车"走势表明视频变现高度依赖单条或少量爆款笔记，缺乏可持续的、可复制的视频变现模式。

6.2 根因分析

  • 爆款依赖症：5月视频GPM冲高至77.55，极可能由个别高转化视频笔记驱动。一旦爆款生命周期结束，视频GPM随之崩塌。
  • 视频内容规模化困难：与图文笔记相比，视频制作成本高、产出频率低，难以在短期内大量铺量形成稳定产出。
  • 本周归零：本周整体曝光仅9万，视频笔记可能完全没有获得有效分发，或本期未产出视频内容。
  • 趋势警示：5月视频GPM（77.55）远超整体GPM（38.85），说明视频在当时是"效率放大器"；但6月骤降后整体GPM也被拖累。

6.3 视频GPM演化趋势

建议：视频笔记应当作为内容矩阵的"效率前锋"而非"主力部队"。建议控制视频笔记占比在20%~30%，同时建立视频内容模板和制作SOP，降低单条视频的"成败权重"，避免因单条爆款退潮导致整体数据大幅波动。

7.2 商品定价策略分析

Q2客单价走势分析：

  • ¥69价格带是核心"爆款价位"：几乎所有高GPM笔记的售价都围绕¥69（虾仁、三文鱼小包装），说明该价格带精准匹配了"一人食减脂"用户群体的心理价位
  • ¥300价格带（整条三文鱼）客单价更高，但订单量较少，适合作为"高阶/礼品"定位
  • 客单价波动：3月¥88.33 → 4月¥124.15（高价值商品成交增加）→ 5月¥90.51 → 6月¥104.92 → 本周¥95.21，整体在¥88~¥124区间震荡

7.3 "低曝光高GPM"与"高曝光低GPM"对比

值得关注的矛盾：
低曝光高GPM笔记（如小柚的2,844.44 GPM）曝光仅45次即产生成交，说明内容精准触达了高购买意图的用户，但受限于曝光量未能扩大成交规模。

高曝光低GPM笔记（如菲菲的347,388曝光、GPM仅22.38）虽然获得了海量曝光，但转化效率极低，内容吸引来的可能是不相关用户。

核心启发：当前运营的核心矛盾不是"曝光不够"，而是"精准曝光不足"。应优先对高GPM笔记进行定向投流放大，而非普遍增加所有笔记的曝光量。

7.4 内容主题聚类

全Q2笔记内容主题呈现三大方向：

  • 省钱认知（占比最高）：以"区区69块""低估了购买力""够吃一周"为核心的性价比内容，是GPM表现最好的方向
  • 对比种草：以"某姆vs这家""以前吃XX现在换这个"为结构的竞品对比内容
  • 减脂日常：以"控体期""减脂餐""爬坡后"为场景的生活方式内容，引流效果强但变现转化偏低

8 全Q2结论与行动建议

8.1 三大核心结论

结论一：GPM震荡上行但基础脆弱
Q2整体GPM从3月37.19上行至本周131.93（+255%），但过程充满波动（4月54.10→5月38.85→6月34.62），且本周飙升主要由分母效应驱动。真正的变现效率改善需要在曝光量恢复到正常水平时验证。
结论二：创作者梯队初具雏形但分化加速
四位创作者逐步形成"一超（鲜水湾）多强（小柚/菲菲/婉婉）"的格局。鲜水湾Q2均值GPM达86.55，远超其他三位。但四位创作者的5期GPM均值均未超过100（除本周异常值外），整体变现效率仍有较大提升空间。
结论三：变现模式从"广撒网"向"精准打击"转型
3~5月以扩大曝光为主要策略（曝光量从60万→205万），但6月起曝光收缩后GPM反而回升（34.62→131.93），说明当前阶段"做少做精"比"做多做广"更有效。"69元虾仁"类高GPM笔记的共性特征是转化路径极短、受众极度精准。

8.3 预期效果量化

如上述建议在Q3初（7~8月）全面落实，预期可实现：

  • 整体GPM稳定在50~70区间（排除极端分母效应干扰）
  • 月均GMV恢复至¥60,000~¥80,000水平（回到4~5月水平）
  • 视频GPM稳定在40+，不再出现归零情况
