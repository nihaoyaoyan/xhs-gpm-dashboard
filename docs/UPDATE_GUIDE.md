# 小红书GPM数据看板 · 每日更新流程

## 更新步骤

**Step 1**：从商家后台导出最新数据
- `商品笔记数据-(YYYY年MM月).xlsx`（含GMV）
- `笔记分析_笔记维度数据_xxx.xlsx`（含曝光量）

**Step 2**：把两个文件传给我，说：
> "分析今天的数据，更新GPM看板"

**Step 3**：自动完成以下流程
1. 数据清洗与合并（通过笔记ID关联）
2. 计算GPM及各项指标
3. 生成月报级仪表盘+报告
4. 推送更新到 GitHub Pages

---

## 数据文件命名约定

建议统一文件名格式，便于自动化处理：
- `商品笔记_YYYYMM.xlsx` — 含GMV的表格
- `笔记维度_YYYYMM.xlsx` — 含曝光量的表格
